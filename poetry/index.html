# 诗和远方

# 记录一下好的文案和句子

且将新火试新茶，诗酒趁年华。 ----《望江南-超然台作》

且挨过三冬四夏，暂受些此痛苦，雪尽后再看梅花。----《百字铭》

最是人间留不住，朱颜辞镜花辞树。 ----《蝶恋花》

玲珑骰子安红豆，入骨相思知不知。----《杨柳枝》

我见青山多妩媚，料青山见我如是。

尝遍人间甘辛味，言外冷暖我自知。----夏目漱石《我是猫》

少年不识愁滋味，爱上层楼。爱上层楼，为赋新词强说愁。
如今识尽愁滋味，欲说还羞。欲说还羞，却道天凉好个秋。

同时天涯沦落人，相逢何必曾相识。
